# Roblox Song Codes Finder

A Next.js web application that showcases working Roblox song (audio) codes, making it easy for players to find and use music IDs in Roblox games. The project collects and displays updated Roblox song codes in a simple, fast, and user-friendly interface.

## Features

- Displays working Roblox song and audio codes
- Simple and easy-to-use interface
- Fast performance using Next.js
- Responsive layout for all devices

## Tech Stack

- Framework: Next.js
- Language: TypeScript
- Styling: CSS / Tailwind (if applicable)
- Runtime: Node.js

## Data Source

Song code data is gathered and referenced from:

https://github.com/rupam2234/Roblox-Song-Codes

Credits to the original contributors for compiling and updating the song codes.

## Getting Started

1. Clone the repository

```bash
git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
```
2. Install dependencies

```bash 
npm install
# or
yarn install
# or
pnpm install
```

3. Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev

```

Project Structure : app/
  page.tsx        # Main page
  components/     # UI components
  styles/         # Styling files
  data/           # Roblox song codes (if applicable)
